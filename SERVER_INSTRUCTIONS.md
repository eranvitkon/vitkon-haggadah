# 🌐 הגדה של משפחת ויתקון - שרת Real-Time

## מה זה כולל?

עכשיו ההגדה שלך עובדת עם **שרת** שמאפשר:
- ✅ **ראיית משתמשים אחרים בזמן אמת** - כל המשפחה רואה מי מחובר ובאיזה עמוד הם נמצאים
- ✅ **שיתוף תמונות בזמן אמת** - כל תמונה שמישהו מעלה מופיעה אצל כולם מיד!
- ✅ **צילום ישיר מהמצלמה** - אפשר לצלם ישירות מהמכשיר
- ✅ **סנכרון אוטומטי** - אם מישהו מתנתק ומתחבר שוב, הכל מסונכרן

---

## 📦 הקבצים שקיבלת:

1. **package.json** - הגדרות השרת
2. **server.js** - קוד השרת (Node.js)
3. **haggadah-server.html** - ההגדה המחוברת לשרת

---

## 🚀 אופציה 1: הרצה מקומית (לבדיקה)

### דרישות:
- Node.js מותקן במחשב (הורד מ-https://nodejs.org)

### שלבים:

1. **פתח Terminal/CMD** בתיקייה עם הקבצים

2. **התקן תלויות:**
```bash
npm install
```

3. **הרץ את השרת:**
```bash
npm start
```

4. **פתח בדפדפן:**
```
http://localhost:3000
```

5. **בדוק עם כמה מכשירים:**
   - פתח את אותו הקישור במספר טאבים/מכשירים
   - כל אחד יכול להתחבר עם שם ואווטר שונה
   - תראה את כולם בזמן אמת!

---

## 🌍 אופציה 2: פריסה באינטרנט (לשימוש אמיתי)

### A. Render.com (בחינם! מומלץ!)

1. **צור חשבון** ב-https://render.com

2. **New → Web Service**

3. **חבר ל-GitHub:**
   - צור repository ב-GitHub
   - העלה את 3 הקבצים: `package.json`, `server.js`, `haggadah-server.html`

4. **הגדרות:**
   - **Name:** `vitkon-haggadah`
   - **Environment:** Node
   - **Build Command:** `npm install`
   - **Start Command:** `npm start`
   - **Instance Type:** Free

5. **Deploy!**
   - תקבל URL כמו: `https://vitkon-haggadah.onrender.com`
   - שתף את הקישור עם המשפחה!

**חשוב:** גרסה חינמית ישנה אחרי 15 דקות חוסר פעילות. הטעינה הראשונה תקח כ-30 שניות.

---

### B. Railway.app (גם בחינם!)

1. **צור חשבון** ב-https://railway.app

2. **New Project → Deploy from GitHub**

3. **בחר את ה-Repository** שלך

4. **Railway יזהה אוטומטית** שזה Node.js

5. **קבל URL** ושתף!

---

### C. Glitch.com (הכי קל!)

1. **לך ל-https://glitch.com**

2. **New Project → hello-express**

3. **החלף את הקבצים:**
   - מחק את `server.js` הקיים
   - העלה את `package.json`, `server.js`, `haggadah-server.html` שלך

4. **לחץ על Show → In a New Window**

5. **קבל URL** לשיתוף!
